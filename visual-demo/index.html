Symbolic Music Benchmark Visual Demo

This dashboard publishes benchmark outputs, run comparisons, and sample cases from the current repository state. It is designed for sharing results and inspecting benchmark behavior across tasks, tokenizers, and available model runs.

This page is static. To execute the benchmark locally, use benchmark_web.py or bash scripts/run_visual_demo_workflow.sh, then rebuild the assets with python3 scripts/build_visual_demo_site.py.
